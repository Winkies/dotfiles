# Emacs configuration

Just my configuration files for Emacs.

### Installation
On your home (~/)
~~~{.sh}
git clone https://github.com/Winkies/.emacs.d.git
~~~

### Emacs version
* 25.5.1

### Dependencies
* auto-complete
* auto-complete-c-headers
* auto-package-update
* dash
* iedit
* linum-relative
* neotree
* popup
* yasnippet
* zenburn-theme

### Installation dependencies
Normally, they will be installed automatically by a snippet
present in init.el.
